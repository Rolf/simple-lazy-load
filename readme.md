### Ленивая загрузка  
Установка не через npm, простое подключение. Если нужно, импортим в сборщим и собираем.  
#### Как пользоваться  
Исходная картинка:  
````
<img src='path/to/images.jpg'/>
````
Как должно быть:  
````
<img src='javascript:;' data-src='path/to/images.jpg' class='js-lazy'/>
````
